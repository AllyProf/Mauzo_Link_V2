Click the buttons below to see the toast notifications in action!

1. Basic Toast Notifications

Simple notifications with just a message

showToast('success', 'Operation completed successfully!'); showToast('error', 'Something went wrong!'); showToast('warning', 'Please check your input'); showToast('info', 'New update available');

2. Toast with Custom Titles

Notifications with both title and message

showToast('success', 'Your profile has been updated', 'Profile Updated'); showToast('error', 'Please fill all required fields', 'Validation Error'); showToast('warning', 'Your session will expire soon', 'Session Warning');

3. Toast with Custom Duration

Control how long the notification stays visible

showToast('info', 'This will disappear in 1 second', 'Quick Message', 1000); showToast('info', 'This will stay for 5 seconds', 'Normal Message', 5000); showToast('warning', 'This will stay for 10 seconds', 'Important Message', 10000);

4. Real-world Use Cases

Common scenarios in your application

// Login Success showToast('success', 'Welcome back, John Doe!', 'Login Successful'); // Save Operation showToast('success', 'Your changes have been saved', 'Saved'); // Delete Confirmation showToast('success', 'Item deleted successfully', 'Deleted'); // Validation Error showToast('warning', 'Email field is required', 'Validation Error');

5. Multiple Sequential Toasts

SweetAlert2 automatically queues multiple toasts

showToast('info', 'Processing your request...', 'Step 1'); setTimeout(() => showToast('warning', 'Validating data...', 'Step 2'), 500); setTimeout(() => showToast('success', 'Operation completed!', 'Step 3'), 1000);

6. AJAX Integration Example

Simulate an AJAX request with toast feedback

$.ajax({ url: '/api/save-data', method: 'POST', data: formData, success: function(response) { showToast('success', response.message, 'Success!'); }, error: function(xhr) { showToast('error', xhr.responseJSON.message, 'Error!'); } });
